# BoatMatey App Icon Concepts

## Concept 1: Compass Rose + Wave
**Description:** A clean compass rose with cardinal directions, overlaid with a subtle wave pattern at the bottom.

**Colors:**
- Compass: Deep teal (#0d9488) with navy accents
- Wave: Lighter teal (#14b8a6) with seafoam highlights
- Background: White or subtle gradient

**Style:** Minimal, geometric, premium yacht aesthetic

## Recommended Implementation

For the app icon, I recommend **Concept 1 (Compass Rose + Wave)** as it:
- Clearly represents navigation and marine themes
- Is instantly recognizable
- Works well at small sizes
- Matches the in-app logo design
- Feels premium and professional

The icon should be designed at multiple sizes:
- 1024x1024 (App Store)
- 512x512 (Google Play)
- 192x192 (Android)
- 180x180 (iOS)

All concepts use the same color palette as the UI for brand consistency.
